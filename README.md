## ✦ About Me

```yaml
name:      Diezels
role:      Developer @ MCServerHost
building:  The MCServerHost ecosystem — mcsh.tools & mcshclient
stack:     Kotlin · Java · Minecraft · Infrastructure
location:  United Kingdom
```

I build the tooling and client software that power **MCServerHost** — a Minecraft server
hosting platform. My work spans developer tools, the MCServerHost client, and custom
Minecraft plugins, mostly written in **Kotlin**.

💬 Find me and the community on [Discord](https://discord.gg/mcserverhost).
